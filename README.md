# ✨ Mini Story Generator

An interactive web application that generates short two-paragraph stories based on user-selected genres.

## 💡 About the Project
This project allows users to choose a genre and instantly generate a mini story. It combines creativity with JavaScript logic to create a fun and engaging experience.

## 🚀 Features
- Multiple genres (Romance, Mystery, Fantasy, Adventure, Sci-Fi)
- Generates two-paragraph stories
- Clean and aesthetic user interface
- Interactive and user-friendly design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🎯 Purpose
This project was built to practice JavaScript and create a creative storytelling web application.

## 🌐 How to Use
1. Select a genre  
2. Click "Generate Story"  
3. Read your randomly generated story  

## 📌 Future Improvements
- Add more genres and stories  
- Add typing animation effect
